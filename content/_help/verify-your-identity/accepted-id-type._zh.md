  - "**美国护照。**目前还不接受使用护照进行[亲身验证](/zh/help/verify-your-identity/verify-your-identity-in-person/)。如果你没有看到使用护照验证身份的选项，请使用驾照或州 ID，或者稍后再次尝试验证你的身份。"
dont_list:
  - 美国护照卡和非美国护照
  - 军人 ID
  - 枪支执照或许可证
  - 过期的 ID 或延期文件
  - 纸质或临时 ID
